Windows 8.1 : käytön perusteet
Tässä opinnäytetyössä suunniteltiin ja toteutettiin Windows 8-alustalla toimiva laitteen etähallintaohjelma. Työ suunniteltiin ja toteutettiin käyttäen Windows 8 Consumer Preview-nimeä kantavaa esittelyversiota sekä Visual Studio 11 Betaa. Ohjelman tarkoituksena oli raportoida laitteen tietoja verkkopalveluun, sekä suorittaa verkkopalvelun määräämiä tehtäviä. Sovelluksen käyttöliittymä rakennettiin XAML-kielellä ja ohjelman toiminta toteutettiin C#-kielellä. Ohjelmoinnissa käytettiin WinRT-kirjastoja sekä .NET-ohjelmistokehystä. Työn käyttöliittymä ja toiminta jaettiin MVVM-arkkitehtuurimallin mukaisiin osioihin, jolloin käyttöliittymä ja toiminnallinen koodi pystyttiin erottelemaan toisistaan. Sovellus suuniteltiin siten, että se pystyttiin jakamaan modulaarisiin osiin jatkokehitystä varten. Ohjelmassa käytettiin WinRT:n BackgroundTask luokkaa tehtävien suorittamiseen taustalla. Projektissa saatiin tehtyä toimiva ohjelma. Metron asettamien rajoitteiden vuoksi, ohjelmalla pystyy suorittamaan vain paikannus ja tilanpäivitys toiminnot.
Windows 8 : käytön perusteet
Tässä opinnäytetyössä suunniteltiin ja kehitettiin Agenteq Solutions Oy:n käyttöön huoneistotarkastussovellus Windows 8 -taulutietokoneille. Toimeksiannon lähtökohtana oli kehittää huoneistojen tarkastajille työkalu, jolla he voivat suorittaa työnsä entistä tehokkaammin ja niin, että he saavat työhönsä siitä lisäarvoa. Haastetta kehitysprosessiin toivat uusi Windows 8 -käyttöjärjestelmä ja uusien Windows Store -sovellusten kehittäminen. Sovellusta kehitettiin iteratiivisesti. Sovelluskehityksessä ei voitu tiukkaan määrittää tapaa, jolla haluttu toiminto toteutetaan, koska uusien sovellusten kehitys vaati opiskelua. Projektissa oli mukana pilottiasiakkaita, jotka pääsivät mukaan kehitysprosessiin. Kehitysprosessi onnistui ja tuloksena saimme ensimmäisen tuotantoversion Tampuuri Tablet tuotteesta huhtikuun 2013 aikana. Tampuuri Tablet on saanut hyvän vastaanoton asiakkailta.

Tiivistelmä: Tämä opinnäytetyö esittelee Microsoft Windows 8 -käyttöjärjestelmän uusia ominaisuuksia ja parannuksia Microsoftin edellisiin käyttöjärjestelmiin. Opinnäytetyön tavoitteena on hahmottaa Windows 8 -käyttöjärjestelmän näkyvimpiä udistuksia ja sitä, mitä ne tarjoavat käyttäjille. Opinnäytetyössä käydään läpi Windows 8:n keskeiset käyttöliittymä-, ohjelmisto- ja toimintouudistukset sekä mitä tulevaisuudessa voidaan odottaa Windowsilta. Työssä tullaan käymään läpi myös Windows 8:n asennus ja sen eri vaiheet. Työ tehtiin Metropolia Ammattikorkeakoululle.  Windows 8:aa ei ollut julkaistu vielä, kun tätä opinnäytetyötä alettiin tehdä. Osa opinnäytetyöstä perustuu artikkeleihin ja Windows 8 -betaversioihin, ennen kun Windows 8 julkaistiin. Insinöörityön aikana Windows 8 kuitenkin julkaistiin ja se muutti työn tarkoitusta hieman. Työn lopussa tultiin tulokseen, että Microsoft on tehnyt Windows 8:sta uusimman lippulaivansa ja Windows 8:n julkaiseminen ensimmäisenä kosketusnäyttöteknologiaan pohjautuvana käyttöjärjestelmänä tulee vaikuttamaan pitkälle tulevaisuuteen.
